# 四组消融实验配置说明

基于当前实验配置文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

四组实验只修改以下位置，其余配置保持完全一致：

```yaml
model:
  vision_model:
    use_target_point_camera_attention: ...

data_module:
  base_dataset:
    route_as: ...
    use_lmdrive_commands: false

name: ...
```

为保证公平性，四组实验应保持相同的数据集、训练轮数、batch size、学习率和随机种子。

---

## 实验 A：无导航条件、无六视角注意力、无LG注意力监督

### 作用

作为不使用导航条件和目标点注意力的基础模型。

### 修改位置

文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

修改为：

```yaml
model:
  vision_model:
    use_target_point_camera_attention: false

data_module:
  base_dataset:
    route_as: none
    use_lmdrive_commands: false

name: simlingo_lg_no_navigation_no_attention_seed_9876
```

---

## 实验 B：HLC 导航、无六视角注意力、无LG注意力监督

### 作用

验证传统高层导航指令 HLC 对驾驶预测的作用。

### 修改位置

文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

修改为：

```yaml
model:
  vision_model:
    use_target_point_camera_attention: false

data_module:
  base_dataset:
    route_as: command
    use_lmdrive_commands: false

name: simlingo_lg_hlc_no_attention_seed_9876
```

说明：

```text
route_as: command
```

只使用代码中由 CARLA command 映射得到的 HLC 文本，不使用 LMDrive 增强命令。

---

### 作用

### 修改位置

文件：

```text

```

修改为：

```yaml
model:
  vision_model:
    use_target_point_camera_attention: false

data_module:
  base_dataset:
    route_as: target_point
    use_lmdrive_commands: false

name: simlingo_lg_target_point_no_attention_seed_9876
```

---

## 实验 D：TARGET_POINT 导航、启用六视角注意力、无LG注意力监督

### 作用

完整模型，用于验证目标点引导的六视角相机注意力带来的额外提升。

### 修改位置

文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

修改为：

```yaml
model:
  vision_model:
    use_target_point_camera_attention: true

data_module:
  base_dataset:
    route_as: target_point
    use_lmdrive_commands: false

name: simlingo_lg_target_point_attention_seed_9876
```

---

## 四组实验汇总

| 实验 | `route_as` | `use_target_point_camera_attention` | `use_lmdrive_commands` |
|---|---|---:|---:|
| A：无导航、无注意力 | `none` | `false` | `false` |
| B：HLC、无注意力 | `command` | `false` | `false` |
| C：TARGET_POINT、无注意力 | `target_point` | `false` | `false` |
| D：TARGET_POINT、启用注意力 | `target_point` | `true` | `false` |

## 不应修改的共同配置

四组实验应保持以下内容一致：

```yaml
model:
  lr: 3e-5
  predict_route_as_wps: true
  speed_wps_mode: 2d

data_module:
  batch_size: 6
  num_workers: 8

max_epochs: 15
val_every_n_epochs: 2
seed: 9876
```

训练数据、验证数据、LG 监督方式、图像增强方式以及其他网络结构也应保持一致。
