# 五组消融实验配置说明

基于当前实验配置文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

五组实验只修改以下位置，其余配置保持完全一致：

```yaml
model:
  use_lg_camera_attention_supervision: ...
  lg_camera_attention_loss_weight: 0.05

  vision_model:
    use_target_point_camera_attention: ...

data_module:
  base_dataset:
    route_as: ...
    use_lmdrive_commands: false

name: ...
```

为保证公平性，五组实验应保持相同的数据集、训练轮数、batch size、学习率、随机种子、LG语言监督方式和LG轨迹监督方式。

---

## 实验 A：无导航条件、无六视角注意力、无LG注意力监督

### 作用

作为不使用导航条件、目标点六视角注意力和LG注意力监督的基础模型。

### 修改位置

文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

修改为：

```yaml
model:
  use_lg_camera_attention_supervision: false
  lg_camera_attention_loss_weight: 0.05

  vision_model:
    use_target_point_camera_attention: false

data_module:
  base_dataset:
    route_as: none
    use_lmdrive_commands: false

name: simlingo_lg_no_navigation_no_attention_seed_9876
```

说明：

- `route_as: none`：不向模型提供HLC或TARGET_POINT导航条件；
- `use_target_point_camera_attention: false`：关闭目标点引导的六视角注意力；
- `use_lg_camera_attention_supervision: false`：不计算LG相机注意力辅助损失；
- 此时 `lg_camera_attention_loss_weight` 不参与实际计算，保留为统一配置值。

## 实验 B：HLC导航、无六视角注意力、无LG注意力监督

### 作用

验证传统高层导航指令HLC对驾驶预测的作用。

### 修改位置

文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

修改为：

```yaml
model:
  use_lg_camera_attention_supervision: false
  lg_camera_attention_loss_weight: 0.05

  vision_model:
    use_target_point_camera_attention: false

data_module:
  base_dataset:
    route_as: command
    use_lmdrive_commands: false

name: simlingo_lg_hlc_no_attention_seed_9876
```

说明：

```text
route_as: command
```

只使用代码中由CARLA command映射得到的HLC文本，不使用LMDrive增强命令。

---

## 实验 C：TARGET_POINT导航、无六视角注意力、无LG注意力监督

### 作用

验证TARGET_POINT本身相较于无导航条件和HLC导航带来的作用。

### 修改位置

文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

修改为：

```yaml
model:
  use_lg_camera_attention_supervision: false
  lg_camera_attention_loss_weight: 0.05

  vision_model:
    use_target_point_camera_attention: false

data_module:
  base_dataset:
    route_as: target_point
    use_lmdrive_commands: false

name: simlingo_lg_target_point_no_attention_seed_9876
```

说明：

- 模型接收两个TARGET_POINT作为导航条件；
- 不启用六视角相机注意力；
- 不使用LG相机注意力标签进行显式监督。

---

## 实验 D：TARGET_POINT导航、启用六视角注意力、无LG注意力监督

### 作用

验证目标点引导的六视角相机注意力相对于实验C带来的额外提升。

### 修改位置

文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

修改为：

```yaml
model:
  use_lg_camera_attention_supervision: false
  lg_camera_attention_loss_weight: 0.05

  vision_model:
    use_target_point_camera_attention: true

data_module:
  base_dataset:
    route_as: target_point
    use_lmdrive_commands: false

name: simlingo_lg_target_point_attention_seed_9876
```

说明：

- 模型使用TARGET_POINT导航；
- 模型内部启用目标点引导的六视角相机注意力；
- 相机权重只通过原有语言和驾驶任务间接学习；
- 不使用LG生成的 `camera_attention_target` 进行显式监督。

---

## 实验 E：TARGET_POINT导航、启用六视角注意力、启用LG注意力监督

### 作用

在实验D基础上，引入LG反事实因果分析生成的六视角相机注意力软标签，验证LG视觉证据监督是否能够进一步改善六视角权重分配以及最终语言和驾驶预测性能。

### 修改位置

文件：

```text
simlingo_training/config/experiment/simlingo_lg_seed1.yaml
```

修改为：

```yaml
model:
  use_lg_camera_attention_supervision: true
  lg_camera_attention_loss_weight: 0.05

  vision_model:
    use_target_point_camera_attention: true

data_module:
  base_dataset:
    route_as: target_point
    use_lmdrive_commands: false

name: simlingo_lg_target_point_attention_lg_supervision_seed_9876
```

说明：

- 使用TARGET_POINT导航；
- 启用目标点引导的六视角相机注意力；
- 对 `camera_attention_valid: true` 的LG样本，使用 `camera_attention_target` 计算相机注意力辅助损失；
- 对 `camera_attention_valid: false` 的样本，不计算相机注意力辅助损失，但仍正常计算语言、waypoint和route等原有损失；
- `lg_camera_attention_loss_weight: 0.05` 表示LG相机注意力辅助损失的权重为0.05。

## 五组实验汇总

| 实验 | `route_as` | `use_target_point_camera_attention` | `use_lg_camera_attention_supervision` | `lg_camera_attention_loss_weight` | `use_lmdrive_commands` |
|---|---|---:|---:|---:|---:|
| A：无导航、无注意力、无LG监督 | `none` | `false` | `false` | `0.05`（不生效） | `false` |
| B：HLC、无注意力、无LG监督 | `command` | `false` | `false` | `0.05`（不生效） | `false` |
| C：TARGET_POINT、无注意力、无LG监督 | `target_point` | `false` | `false` | `0.05`（不生效） | `false` |
| D：TARGET_POINT、启用注意力、无LG监督 | `target_point` | `true` | `false` | `0.05`（不生效） | `false` |
| E：TARGET_POINT、启用注意力、启用LG监督 | `target_point` | `true` | `true` | `0.05` | `false` |

---

## 主要对比关系

### A vs. B

验证HLC导航相对于无导航条件的作用。

### B vs. C

比较传统HLC导航与TARGET_POINT连续几何导航的差异。

### C vs. D

验证目标点引导的六视角相机注意力本身是否有效。

### D vs. E

验证LG生成的因果视觉证据监督是否能够在已有六视角注意力结构上带来进一步提升。

其中，实验D和实验E之间仅允许改变：

```yaml
model:
  use_lg_camera_attention_supervision: false  # 实验D
```

与：

```yaml
model:
  use_lg_camera_attention_supervision: true   # 实验E
```

其他配置必须保持完全一致。

---

`lg_camera_attention_loss_weight` 在五组实验中统一写为 `0.05`，但只有实验E启用LG注意力监督时才实际生效。
